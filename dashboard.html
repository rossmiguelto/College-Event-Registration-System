Visual 1 — Registrations by Event (Bar Chart)

Annotation: A horizontal bar chart is appropriate because event names are text labels (categories), not numbers. Bars make it immediately clear which event has the highest registration count. Data is pulled live from the Registrations table using SQL GROUP BY.

Visual 2 — Daily Registration Volume (Line Chart)

Annotation: A line chart is appropriate for time-series data because it conveys trend and continuity. Connecting points day-by-day makes growth or decline immediately visible. The filled area below the line adds emphasis to registration volume. Data is grouped by DATE(reg_date) in SQL and rendered in Python via matplotlib.
